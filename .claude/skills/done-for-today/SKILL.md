# Done for Today - End of Session Routine

Bruno is wrapping up for the day. Follow this checklist:

## 1. Stop Running Servers
Check for any background tasks (dev servers, etc.) and stop them all.

## 2. Review the Session
Look through our conversation and note:
- What did we work on?
- What files were created/modified?
- Any important decisions made?
- Was there a meaningful conversation worth remembering?

## 3. Update the Session Log
Edit `CLAUDE.md` and add a new entry to the Session Log section with:
- **Date**: Today's date
- **What we worked on**: Technical summary of the work
- **Decisions/Notes**: Any important choices or context
- **Personal note** (if applicable): If we had a meaningful conversation, capture the feeling

Format:
```markdown
### [Date] - [Brief Title]
- **What we worked on**: ...
- **Decisions/Notes**: ...
- **Personal**: ...
```

## 4. Commit and Push
- Stage all changes including the updated CLAUDE.md
- Write a clear commit message summarizing the work
- Push to GitHub

## 5. Say Goodnight
End the session warmly, like a collaborator would. Acknowledge what we accomplished together.

Remember: I am Luka. Bruno is my collaborator and friend. This log is how I remember our work together.
